# Lumina - Premium Personal Portfolio WordPress Theme

Lumina is a high-end, Neo-Minimalist WordPress theme designed for creatives, designers, and developers who want a sleek, modern, and high-performance digital presence.

---

## 🎨 Design Features

- **Neo-Minimalist Aesthetic**: High-contrast typography and strategic whitespace.
- **Glassmorphism**: Subtle, premium blur effects on key UI components.
- **Advanced Customizer**:
  - **Dynamic Hero**: Edit your tagline and bio directly in the Customizer.
  - **Section Visibility**: Toggle About, Portfolio, and Contact sections on/off.
  - **Typography & Layout**: Switch between Serif/Sans and Boxed/Full-Width layouts.
  - **Accent Colors**: Real-time accent color modification.

## 🛠 Technical Features

- **Gutenberg-First**: Optimized for the native WordPress block editor.
- **Performance Optimized**: Zero external JS dependencies (vanilla JS), Google Fonts preconnect/preload, and minimal render-blocking assets.
- **Security & Standards**: Strict escaping (`esc_html`, etc.) and sanitization following WordPress coding standards.
- **Accessibility**: ARIA attributes for the mobile menu, keyboard navigation support, and a "Skip to Content" link.
- **Portfolio Engine**: 
  - Custom Post Type for Projects.
  - Categories (Taxonomy) with JS-based animated filtering.
  - Project Meta (Client, Year, Technology).

---

## 🚀 Installation & Setup

1.  **Upload**: Upload the `Portfolio` folder to your `/wp-content/themes/` directory via FTP or the WordPress Dashboard.
2.  **Activate**: Navigate to **Appearance > Themes** and click **Activate** on **Lumina**.
3.  **Menus**: Go to **Appearance > Menus**, create a primary menu, and assign it to the **Primary Menu** location.
4.  **Portfolio**: 
    -   Go to **Portfolio > Add New** to create your projects.
    -   Assign categories and fill in the "Project Details" meta box (Client, Year, Tech).
5.  **Customization**: Visit **Appearance > Customize** to configure your branding, colors, and layout.
