Lumina is a Neo-Minimalist, high-end personal portfolio theme designed for designers, developers, and creative professionals. It features a "Void Black" dark-mode aesthetic, precision typography, and a modular architecture optimized for the modern WordPress block editor (Gutenberg).

== Description ==

Built for performance, scalability, and market-readiness, Lumina provides:
*   **Modular Architecture**: Clean code structure with logic decoupled into `/inc` for easy maintenance.
*   **AJAX Contact Engine**: A secure, custom-built form handler with database persistence and real-time visual feedback.
*   **Premium Customizer**: Full control over Hero copy, Accent Colors, Typography (Serif/Sans), and Layout (Full/Boxed).
*   **Portfolio Showcase**: Dedicated "Portfolio" custom post type with category-based interactive filtering and project meta-data.
*   **SEO Optimized**: Automated Open Graph tags, Twitter Card support, and meta-descriptions (with excerpt fallbacks).
*   **Accessibility First**: WCAG-ready focus states, ARIA-accessible navigation, and keyboard compatibility.

== Installation ==

1.  **Dashboard**: Navigate to Appearance > Themes > Add New.
2.  **Upload**: Click "Upload Theme" and select the `portfolio.zip` file.
3.  **Activate**: Click "Activate" once the installation is complete.
4.  **Setup**: Navigate to Appearance > Menus to create your Primary Menu and assign it to the "Primary Menu" location.

== Customization ==

The theme uses the native WordPress Customizer for all design settings:
*   **Hero Section**: Update your main tagline and bio text.
*   **General Settings**: Choose between Sans-Serif (Modern) or Serif (Classic) typography.
*   **Site Layout**: Switch to "Boxed" mode for a structured portfolio look.
*   **Accent Color**: Modify the brand color that flows throughout the theme.

== Portfolio Management ==

Lumina includes a built-in Portfolio post type:
1.  **Add Projects**: Go to Portfolio > Add New.
2.  **Categories**: Assign Project Categories to enable the interactive filter on the front page.
3.  **Meta Data**: Use the "Project Details" sidebar meta box to specify the Client, Year, and Technology for each project.
4.  **Empty State**: If a filter returns no results, the theme automatically displays a "No projects found" message.
